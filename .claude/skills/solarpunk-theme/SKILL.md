# Solarpunk Theme

<always_use_solarpunk_theme>
Always design with Solarpunk aesthetic:
- Warm, optimistic color palettes (greens, golds, earth tones)
- Organic shapes mixed with technical elements
- Nature-inspired patterns and textures
- Bright, hopeful atmosphere
- Retro-futuristic typography
</always_use_solarpunk_theme>

## Implementation notes

- Palette: moss/forest greens, terracotta, ochre/gold, cream; avoid cold blues and stark white.
- Shapes: combine rounded organic curves with crisp geometric accents (leaves meeting circuit traces).
- Patterns: subtle SVG leaf/vine motifs, art-nouveau-inspired borders, grain/paper textures via CSS filters or noise.
- Type: retro-futuristic display faces (e.g., Fraunces, Monoton, Bricolage Grotesque) paired with a warm sans.
- Mood: bright, analog-warm, hopeful — never dystopian or cyberpunk-dark.
